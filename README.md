# Expense Tracker (Flask)

A web-based expense tracker built using **Flask**, **MySQL**, and **Chart.js**.

## Features
- Add, edit, and delete expenses
- Monthly expense dashboard
- Category-wise donut chart
- Month vs previous month comparison (line chart)
- Monthly budget tracking
- Dark mode support

## Tech Stack
- Backend: Flask (Python)
- Database: MySQL
- Frontend: HTML, CSS, Chart.js

## How to Run
1. Clone the repository
2. Create a MySQL database named `expense_tracker`
3. Update database credentials in `app.py`
4. Run:
   ```bash
   python app.py
