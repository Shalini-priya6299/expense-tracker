# Expense Tracker

A simple, lightweight expense tracking web application to help users record, categorize, and visualize personal expenses. Built for ease of use with a clean UI and basic analytics.

## Features

- Add, edit, and delete expenses
- Categorize expenses (e.g., Food, Transport, Utilities)
- View expenses by date range and category
- Summary dashboard with totals and simple charts
- Export expenses to CSV
- Responsive layout for mobile and desktop

## Tech Stack

- Frontend: HTML, CSS, JavaScript (or your chosen framework)
- Backend: (optional) Node.js / Express or any backend of your choice
- Database: (optional) SQLite / PostgreSQL / MongoDB

> Tip: If this is a client-only project, remove backend/database sections.

## Installation

1. Clone the repository:

   git clone https://github.com/Shalini-priya6299/expense-tracker.git
   cd expense-tracker

2. Install dependencies (if applicable):

   npm install

3. Start the development server:

   npm start

4. Open the app in your browser at http://localhost:3000 (or as your app specifies)

## Usage

- Click "Add Expense" to create a new expense entry.
- Use filters to view expenses for a specific date range or category.
- Check the dashboard to see totals and charts for spending trends.
- Export data to CSV from the export button.

## Configuration

- If using a backend, set environment variables in a `.env` file (examples):

  PORT=3000
  DATABASE_URL=your_database_url
